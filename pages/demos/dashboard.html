Nested Dashboard

A real UI — header, sidebar, and a responsive display: grid card area, each card its own nested flex layout — laid out entirely by solveLayout and drawn with absolutely-positioned divs. Resize the window and the solver re-runs over the whole tree. Flip to CSS layout to render the exact same tree with native flex and grid: the badge measures both and confirms they land in the same place.
